schema 參照 cangjie6.schema

* 反查字根 （全部/拼音/倉頡/蝦米 ） `  / `P / `C /`L
* 拼音/倉頡/蝦米字根輸入    P   C  L 
* 二 三 選字   “-”   “.'
* 簡碼 頭 + v + 尾
