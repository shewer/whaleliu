# whaleliu   鯨魚舞倉頡 輸入法
### 原由: 倉頡五、六字典完整，見字拆字，所以才有了轉換輸入法的想法，但是發覺字根鍵位要重背 手指記憶又是一個門檻；於是有了字根轉換的想法。且倉頡系列輸入法都可以延用
### rime 輸入法框架在各系統上都支援不愁變種輸入法變成孤兒
